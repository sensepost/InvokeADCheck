## Installation
InvokeADCheck is available on the [PowerShell Gallery](https://www.powershellgallery.com/packages/InvokeADCheck/).

To Inspect:
```powershell
Save-Module -Name InvokeADCheck -Path <path>
```
To install:
```powershell
Install-Module -Name InvokeADCheck -Scope CurrentUser
```
