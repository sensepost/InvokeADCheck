## Installation
ADChecks is available on the [PowerShell Gallery](https://www.powershellgallery.com/packages/ADChecks/).

To Inspect:
```powershell
Save-Module -Name ADChecks -Path <path>
```
To install:
```powershell
Install-Module -Name ADChecks -Scope CurrentUser
```
